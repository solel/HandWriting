딥러닝 계산 흐름

위에서 직접 그린 숫자는 아래 흐름을 따라 계산됩니다. 28×28 픽셀을 정규화한 뒤 입력층에 넣고, 은닉층과 출력층을 거쳐 softmax 확률로 결과가 표시됩니다.

1. 손글씨 입력 캔버스에 쓴 숫자가 분류할 대상이 됩니다.
2. 픽셀 변환 그림을 28×28 크기의 작은 픽셀 격자로 바꿉니다.
3. 정규화 픽셀 밝기를 0~1 사이 숫자로 맞춰 계산하기 쉽게 만듭니다.
4. 입력층 28×28 픽셀을 한 줄로 펼친 784개 값이 들어갑니다.
5. 은닉층 선의 위치와 모양 같은 특징을 조합합니다.
6. 출력층 0~9에 해당하는 10개 노드가 값을 냅니다.
7. softmax 확률 출력층의 값을 확률처럼 바꿔 보여줍니다.
