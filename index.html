딥러닝 계산 흐름

위에서 직접 그린 숫자는 아래 흐름을 따라 계산됩니다. 28×28 픽셀과 784개 입력값은 입력층으로 들어갈 데이터이고, 신경망 계산을 거친 뒤 소프트맥스 확률로 결과가 표시됩니다.

1. 손글씨 입력 캔버스에 쓴 숫자가 분류할 대상이 됩니다.
2. 픽셀 변환 그림을 28×28 크기의 작은 픽셀 격자로 바꿉니다.
3. 784개 입력값 28×28 픽셀을 한 줄로 펼쳐 입력층에 넣습니다.
4. 정규화 픽셀 밝기를 0~1 사이 숫자로 맞춰 계산하기 쉽게 만듭니다.
5. 신경망 계산 은닉층이 특징을 조합하고 출력층 10개 노드가 값을 냅니다.
6. 확률 출력 소프트맥스가 0~9의 값을 확률처럼 바꿔 보여줍니다.

코드 연결

코랩 코드와 연결하기

reshape(..., 784)28×28 픽셀을 한 줄로 펼친 것
/ 255.0픽셀 밝기를 0~1 사이로 바꾼 것
to_categorical(..., 10)정답을 0~9 중 하나로 표시한 것
Dense(100, relu)중간에서 특징을 찾는 은닉층
Dense(10, softmax)0~9 각각의 가능성을 계산하는 출력층
